Title: 🚀 Rocket Launch Data Pipeline

Description: An automated ETL (Extract, Transform, Load) pipeline built with Apache Airflow that tracks upcoming global space launches and archives mission photography.

Tech Stack: Python, Apache Airflow (Bash & Python Operators), Docker, and The Space Devs API.

How it Works: Basically it uses a BashOperator for the initial fetch and a PythonOperator for image processing.
